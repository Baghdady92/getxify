# Security policy

## Supported versions

Only the latest published version of GetXify on [pub.dev](https://pub.dev/packages/getxify) receives security fixes.

| Version | Supported |
|---------|-----------|
| 4.0.0  | ✅ |
| <=3.0.0   | ❌ |

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report security issues by emailing the maintainer directly:

**aniketkhote99@gmail.com**

Include as much detail as possible:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Affected versions
- Any suggested fix, if you have one

You can expect an acknowledgement within **48 hours** and a resolution or status update within **7 days**.

Once the issue is confirmed and a fix is ready, we will:

1. Release a patched version on pub.dev.
2. Publish a security advisory on GitHub.
3. Credit you in the advisory (unless you prefer to remain anonymous).
